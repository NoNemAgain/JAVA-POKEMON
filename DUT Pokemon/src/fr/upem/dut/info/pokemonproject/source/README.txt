Ce projet consiste à reproduire partiellement le célèbre jeu pokémon en binôme. 
Dans ce projet, nous devons coder la partie non aventure de pokémon c'est à dire, majoritairement les combats. 

Ce projet est sous licence GNU GPL 3. 

Pour un fonctionnement optimal, ce projet doit être lancer à partir de Eclipse JAVA JDK11.
